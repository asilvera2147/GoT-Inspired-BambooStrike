# GhostOfTsushimaInspired-BambooStrike
## Technical Overview
This repository contains a clean-room implementation of a timing-based input mechanic built in the Unity Engine. The architecture focuses on precision input indexing, enum-based difficulty, and collision detection to replicate high-fidelity action sequences for technical study.
